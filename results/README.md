# Results

This directory stores output files generated by the scripts and notebooks:

- `route_map.html` - Interactive map showing a single route
- `comparison_map.html` - Map with overlaid routes from all algorithms
- `benchmark_chart.png` - Bar chart comparing algorithm performance

Run the examples to generate these files:

```bash
python examples/find_route.py --from "Oran" --to "Constantine"
python examples/compare_algorithms.py --from "Algiers" --to "Tamanrasset"
```
